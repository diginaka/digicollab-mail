# デジコラボ メール

MailerLite APIラッパー。無料アカウントのAPIキーを設定するだけで、日本語UIから登録者管理・キャンペーン配信・分析までできるメールマーケティングツール。

## 特徴

- **BYOK（Bring Your Own Key）方式** — データは100%ご自身のMailerLiteアカウントに保存されます
- **無料で利用可能** — MailerLite無料プラン（500人 / 月12,000通）で十分始められます
- **日本語UI** — 全画面が日本語に最適化されています
- **二刀流モード** — Supabase連携モード / スタンドアロンモード両対応

## 6つの画面

1. **ダッシュボード** — 登録者数・配信数・開封率・クリック率を一目で
2. **登録者管理** — 検索・追加・編集・グループ分け
3. **グループ＆セグメント** — 配信リストの整理
4. **キャンペーン管理** — 下書き作成・スケジュール配信
5. **配信レポート** — 開封・クリック分析
6. **オートメーション** — ステップメールの一覧と管理

## はじめかた

### 1. インストール

```bash
npm install
```

### 2. 環境変数を設定（Supabase連携モードの場合のみ）

```bash
cp .env.example .env
# .env に Supabase の URL / anon key を記入
```

空のままなら **スタンドアロンモード** で動作します。

### 3. 起動

```bash
npm run dev
```

### 4. MailerLite APIキーを取得

1. [MailerLite](https://app.mailerlite.com) にログイン
2. 左メニュー「Integrations」→「MailerLite API」
3. 「Generate new token」をクリック
4. 発行されたAPIキーをこのツールの「設定」画面に貼り付け

### 5. ビルド

```bash
npm run build
```

## 重要な注意

- **メール本文（HTMLデザイン）のAPI編集は MailerLite Advanced プラン（$18/月）以上が必要です。** 無料プラン・Growing Businessプランでは、メール本文のデザインはMailerLite本家ダッシュボードで行ってください。このツールは「管理・配信・分析」のラッパーです。
- APIキーはユーザー自身のブラウザ/Supabaseに保存されます。デジコラボ側のサーバーには保存されません。
- レート制限：120リクエスト/分（MailerLite側）
